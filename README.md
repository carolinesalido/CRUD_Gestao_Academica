# CRUD Acadêmico em Python

Sistema simples de **CRUD (Create, Read, Update, Delete)** em Python para gerenciar **estudantes**, **disciplinas** e **professores**, com dados salvos em arquivos **JSON**.

## Funcionalidades

- Incluir, listar, editar e excluir cadastros  
- Armazenamento automático em arquivos `.json`  
- Interface via terminal  
- Login simples (sem validação)

## Estrutura

- `CRUD_Academico.py` – Código principal do sistema  
- `estudantes.json` – Dados dos estudantes  
- `disciplinas.json` – Dados das disciplinas  
- `professores.json` – Dados dos professores

1. Execute o arquivo principal:
